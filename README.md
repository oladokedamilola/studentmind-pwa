## 📌 Overview
StudentMind is a Progressive Web Application designed to provide mental health support for university students. It offers anonymous chat-based assistance powered by OpenAI's language models, available 24/7. The application is deployed on PythonAnywhere for reliable, scalable hosting.

## 🎯 Key Features
- Anonymous chat with AI-powered mental health assistant
- Real-time responses using OpenAI models
- Crisis detection and emergency resource escalation
- Personalized coping strategy recommendations
- Mood tracking and self-assessment tools
- PWA capabilities (offline support, push notifications)

## 🛠️ Tech Stack
- **Backend:** Django 4.2, Django REST Framework
- **Frontend:** HTML5, Bootstrap 5, JavaScript
- **AI Engine:** OpenAI API (via GitHub Marketplace)
- **Database:** SQLite (dev) / MySQL (PythonAnywhere)
- **Caching:** Redis (via PythonAnywhere paid plan or Redis Labs free tier)
- **Task Queue:** Celery (using PythonAnywhere task scheduling)
- **Deployment:** PythonAnywhere (shared/paid hosting)
